# 🌍 Interactive Earth Animation

An interactive Earth visualization project built using **HTML** and **CSS** only. This project showcases a beautiful **2D rotating Earth animation** with buttons that allow users to switch between different visual modes, including a simulated **3D Earth effect** created purely with CSS transformations and animations.

## 🚀 Features

* 🌎 Smooth 2D Earth rotation animation
* 🎨 Pure HTML & CSS implementation
* 🔘 Interactive buttons for animation control
* 🌐 CSS-based 3D Earth visualization effect
* 📱 Responsive design for different screen sizes
* ⚡ Lightweight and fast-loading project
* 🖥️ No JavaScript required

## 🛠️ Technologies Used

* HTML5
* CSS3

  * CSS Animations
  * CSS Transforms
  * CSS Transitions
  * CSS Perspective
  * CSS Keyframes

## 🎯 How It Works

1. The Earth is displayed in a rotating 2D animation using CSS keyframes.
2. Users can interact with the provided buttons.
3. Clicking the buttons changes the visual appearance of the Earth.
4. CSS transforms and perspective effects create the illusion of a 3D rotating globe without using JavaScript or external libraries.
